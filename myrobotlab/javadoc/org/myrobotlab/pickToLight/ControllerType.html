      • getModuleList

        public java.util.List<ModuleType> getModuleList()
        Gets the value of the moduleList property.

        This accessor method returns a reference to the live list, not a snapshot. Therefore any modification you make to the returned list will be present inside the JAXB object. This is why there is not a set method for the moduleList property.

        For example, to add a new item, do as follows: 

            getModuleList().add(newItem);

        Objects of the following type(s) are allowed in the list ModuleType
